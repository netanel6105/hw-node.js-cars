Cars API Documentation

Cars Routes

  • Get 10 products by page number: /cars/?page=1 ( GET)
  • Get X product per one page: /cars/?perPage=3 (GET)
  • Search products by name or info /cars/search/?s=bmw (GET)
  • Get prouduct by Category-name: /cars/category/name (GET)
  • Price range /cars/prices/?max=number&min=number&page=number (GET)
  • Search by Sort (e.g - price/company)/cars/?sort=company (GET)
  • Sort form smaller to biger (default: biger-smaller)/cars/?reverse=yes (GET)

Add Product -(only for register user - must send user's auth-token through headers) /cars (POST)

  • Name (required)
  • Price (required)
  • Category (required)
  • Info (required)
  • Img_url
  • User_id (required)

Edit Product -(only for register user - must send user's auth-token through headers) cars/:id (car) (PUT)

  • Name (required)
  • Price (required)
  • Category (required)
  • Info (required)
  • Img_url
  • User_id (required)

Delete Product-(only for register user - must send user's auth-token through headers) cars/:id (car) (DELTEE)

  • Name (required)
  • Price (required)
  • Category (required)
  • Info (required)
  • Img_url
  • User_id (required)

Sign Up: /users (POST)
  • name (required)
  • email (required)
  • password (required)
  • role (required)

Sign In: /users/login (POST)
  • email (required)
  • password (required)
